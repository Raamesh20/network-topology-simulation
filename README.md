# Cisco Enterprise Network Simulation

## Project Overview

This project is a Cisco Packet Tracer-based enterprise network simulation designed to demonstrate practical networking concepts in a real-world organizational environment. The network includes routers, switches, servers, wireless access points, and multiple subnetworks connected through a structured topology.

The project focuses on network communication, infrastructure setup, device connectivity, and essential network services such as DHCP, DNS, web hosting, email services, and wireless networking.

---

# Features

* Multi-router and multi-switch network topology
* DHCP server configuration for automatic IP assignment
* DNS, Web, Email, and FTP server integration
* Wireless network setup using Access Points
* Inter-network communication between multiple departments
* Structured LAN design with subnet segmentation
* End device connectivity testing using ICMP/Ping
* Real-world enterprise-style network infrastructure simulation

# How to Run

1. Install Cisco Packet Tracer
2. Open the `.pkt` project file
3. Run the simulation in Realtime or Simulation mode
4. Test connectivity using Ping or packet simulation tools
